# What is Java?

Java is an OOP, class-based programming language that works on the principle of “Write Once, Run Anywhere” (WORA). It is designed to be platform-independent by using the Java Virtual Machine (JVM), which allows Java programs to run on different operating systems without changing the code.

Java is widely used for building web applications, desktop software, Android apps, enterprise systems, and backend services because of its performance, security, scalability, and large ecosystem.
