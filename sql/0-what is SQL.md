# What is SQL?

**SQL (Structured Query Language)** is the standard language used to define, manage, and query data in relational database systems.

---

## Purpose of SQL

- **Data Definition:** Create and modify the structure of databases and tables.
- **Data Manipulation:** Insert, update, select, and delete data.
- **Data Control:** Manage access and permissions for database objects.
- **Transaction Management:** Ensure safe and consistent execution of operations.

---

## SQL Command Categories

1. **Data Definition Language (DDL):** Defines and alters database structure.
2. **Data Manipulation Language (DML):** Works with the data itself.
3. **Data Control Language (DCL):** Manages access and security.
4. **Transaction Control Language (TCL):** Handles groups of operations as transactions with ACID properties.

---

## Core Concepts

- **Tables:** Store data in rows and columns.
- **Relationships:** Connect tables using keys to maintain integrity.
- **Constraints:** Enforce rules for valid data.
- **Normalization:** Organize data to reduce redundancy.
- **Queries:** Retrieve and manipulate data based on conditions, sorting, and grouping.
